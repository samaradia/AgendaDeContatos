# Lista de Contatos em Java

Projeto simples desenvolvido em Java para praticar Programação Orientada a Objetos (POO) e o uso de `ArrayList`.

## Funcionalidades

- Cadastrar contatos
- Armazenar contatos em uma lista
- Listar nome e telefone dos contatos

## Tecnologias

- Java

## Estrutura

- `Contato.java` → Classe que representa um contato.
- `Main.java` → Classe principal que cria e exibe a lista de contatos.

## Exemplo de saída

```
Lista de contatos:

1. Lika - (11) 97535-3636
2. Sah - (22) 46389-7385
3. Ney - (23) 4793-5735
```
